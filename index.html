✨ What's New in TrackTempest ✨

Version 0.0.0.4 (May 30, 2025)

  • **Accessibility Improvements**: Added a new option to use accessible color schemes for better visibility.
  • **Smaller Dots Option**: Introduced an option for smaller dot sizes on generated maps, ideal for summary views.
  • **Custom Map Upload (Experimental)**: You can now upload your own custom base maps! This feature is still under development.
  • **UI/UX Enhancements**: Minor adjustments to the user interface for a smoother experience.
  • **Bug Fixes**: Squashed a few pesky bugs to improve stability.

Upcoming Features:

  • **Improved Data Export**: Working on more robust data export options.
  • **Interactive Maps**: Planning to introduce interactive map features for dynamic data exploration.
